
# Multiple Myeloma Treatment Response Prediction & Explainability

## Overview
Machine learning project to predict and explain the clinical stage of Multiple Myeloma patients 
using the MM-dataset. The project emphasizes model explainability using SHAP, making predictions 
interpretable for clinical use.

## Dataset
**Multiple Myeloma Dataset (MM-dataset)**  
Guilal R., Bendahmane A.F., Settouti N., Chikh M.E.A., Mesli N.  
University Hospital of Tlemcen, Algeria (2008-2019)  
https://data.mendeley.com/datasets/7wpcv7kp6f/1

- 203 patient records
- 59 clinical features (demographic, hematology, protein tests, imaging, biochemistry)
- 9 target classes based on Durie-Salmon staging system
